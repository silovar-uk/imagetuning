# Feature Matrix

旧版の原本は開発時のローカル参照として保持し、新版にはAI関連コードを移植しない。公開リポジトリでは `legacy/README.md` に移行方針を記録する。

## Phase 1〜6で実装済み

- React + TypeScript + Viteの新規構成
- designmdを踏まえた3カラムUI
- PNG / JPEG / WEBP / PSD読込
- 複数画像のドラッグ＆ドロップ、クリップボード貼り付け
- 画像・図形の選択、移動、表示、ロック
- パン、ホイールズーム、全体表示、100%表示
- ペン、テキスト、四角、円、矢印、吹き出し、カラータグ
- 描画色、線幅、実線・破線
- 画像・図形へのコメント紐付け
- コメント番号の上から順／作成順切替
- Canvas・コメント欄・PNGで番号順を統一
- コメント検索、未入力抽出、Markdownコピー
- 画像・図形をまたぐレイヤー前後移動
- 色置換、2階調化、四辺トリミング
- 画像修正：消去、単色塗り、ぼかし
- ブラシサイズ、不透明度、ぼかし強度
- 画像修正モーダル内Undo、長押しによる修正前比較
- 修正結果の元画像への直接反映とメインUndo
- JSON入出力とschemaVersion 2
- 旧データの安全な移行とAIフィールド無視
- PNG書き出しプレビュー
- PNGの白／黒／透過背景切替
- PNGのコメント番号表示切替
- 未保存警告
- Pull Request時のテスト・型チェック・本番ビルド

## 次フェーズへ移す機能

- 画像からのスポイト操作
- 主要色抽出、色割合分析
- 図形リサイズ用ハンドル
- レイヤーの最前面／最背面への一括移動
- IndexedDBによる自動一時保存
- 復元導線
- 大画像向けBlob履歴とメモリ上限
- Playwrightによる実ブラウザテスト
