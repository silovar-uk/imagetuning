# Feature Matrix

旧版の原本は開発時のローカル参照として保持し、新版にはAI関連コードを移植しない。公開リポジトリでは `legacy/README.md` に移行方針を記録する。

## Phase 1〜5で実装済み

- React + TypeScript + Viteの新規構成
- designmdを踏まえた3カラムUI
- PNG / JPEG / WEBP / PSD読込
- 複数画像のドラッグ＆ドロップ、クリップボード貼り付け
- 画像・図形の選択、移動、表示、ロック
- パン、ホイールズーム、全体表示、100%表示
- ペン、テキスト、四角、円、矢印、吹き出し、カラータグ
- 描画色、線幅、実線・破線
- 画像・図形へのコメント紐付けと番号表示
- レイヤー一覧と選択同期
- 色置換、2階調化、四辺トリミング
- 画像修正：消去、単色塗り、ぼかし
- ブラシサイズ、不透明度、ぼかし強度
- 画像修正モーダル内Undo、長押しによる修正前比較
- 修正結果の元画像への直接反映とメインUndo
- JSON入出力とschemaVersion 2
- 旧データの安全な移行とAIフィールド無視
- 図形・コメント番号を含むPNG書き出し
- 未保存警告

## 次フェーズへ移す機能

- 画像からのスポイト操作
- 主要色抽出、色割合分析
- 図形リサイズ用ハンドル
- コメント番号の位置順／作成順切替
- Markdownコピー
- PNGプレビュー、背景色、番号表示切替
- レイヤー並び替え
- IndexedDBによる自動一時保存
- 大画像向けBlob履歴とメモリ上限
