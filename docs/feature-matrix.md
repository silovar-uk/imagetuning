# Feature Matrix

旧版の原本は開発時のローカル参照として保持し、新版にはAI関連コードを移植しない。公開リポジトリでは `legacy/README.md` に移行方針を記録する。

## STEP 0〜3で実装済み

- React + TypeScript + Viteの新規構成
- designmdを踏まえた3カラムUI
- PNG / JPEG / WEBP / PSD読込
- 複数画像のドラッグ＆ドロップ
- クリップボード画像貼り付け
- Canvas上の画像選択・移動
- パン・ホイールズーム・全体表示・100%表示
- 画像の透明度・表示・ロック・座標変更
- コメント追加・編集・削除・検索
- コメント番号のCanvas表示
- レイヤー一覧と選択同期
- Undo / Redo
- JSON入出力とschemaVersion 2
- 旧データの安全な移行とAIフィールド無視
- PNG書き出し
- 未保存警告

## 次フェーズへ移す機能

- ペン、テキスト、四角、円、矢印、吹き出し、カラータグ
- トリミング、色置換、2階調化、スポイト
- 主要色抽出、色割合分析
- ローカル画像修正（消去・単色塗り・ぼかし）
- モーダル内Undo / Redoと修正前後比較
- コメント番号の位置順採番
- Markdownコピー
- PNGプレビュー、背景色、番号表示切替
- レイヤー並び替え
- 自動一時保存
